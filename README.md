# Bookworm
### An application to help readers manage their home libraries 
### https://vbiblioteca.herokuapp.com/

### Built Using: 
- Express
- EJS
- Method-Override
- bcrypt 
- Express-Sessions 
- MongoDB 
- FlexBox 
- [NYT Developers Books API] (https://developer.nytimes.com/docs/books-product/1/overview)
